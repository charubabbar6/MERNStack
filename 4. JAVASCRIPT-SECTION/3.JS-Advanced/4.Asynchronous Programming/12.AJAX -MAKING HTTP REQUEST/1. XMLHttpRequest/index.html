Making HTTP Requests with XMLHttpRequest

Introduction

XMLHttpRequest is a browser-based API that can be used to send HTTP requests and receive HTTP responses. It plays a crucial role in AJAX programming and allows for asynchronous communication between a client and a server.

Creating an XMLHttpRequest Object

The first step in using the XMLHttpRequest API is to create a new instance of the object. 


const xhr = new XMLHttpRequest();

Initializing a Request

To make an HTTP request, you'll need to initialize it using the open() method. 


xhr.open('GET', 'https://api.example.com/data', true);

Sending the Request

Once the request has been initialized, the send() method is used to send it. 


xhr.send();

Handling Responses

You can use the onload and onerror events to handle successful and failed requests, respectively. 


xhr.onload = function() {
    if (xhr.status >= 200 && xhr.status < 300) {
        console.log('Success:', xhr.responseText);
    } else {
        console.log('Failed:', xhr.status);
    }
};

xhr.onerror = function() {
    console.log('Request error...');
};

Complete Example


const xhr = new XMLHttpRequest();

xhr.open('GET', 'https://api.example.com/data', true);

xhr.onload = function() {
    if (xhr.status >= 200 && xhr.status < 300) {
        console.log('Success:', xhr.responseText);
    } else {
        console.log('Failed:', xhr.status);
    }
};

xhr.onerror = function() {
    console.log('Request error...');
};

xhr.send();

Conclusion

XMLHttpRequest is a powerful tool for making HTTP requests and working with AJAX. While newer technologies like Fetch and Axios are now available, understanding XMLHttpRequest is essential for grasping the underlying mechanics of client-server communication.
